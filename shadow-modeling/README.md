# 3D Urban Shadow Modeling from LiDAR Point Clouds
The work will involve developing a complete processing pipeline to transform massive LiDAR point clouds into a 3D model suitable for the 
computation and visualization of urban shadows.

The objective is to design algorithms capable of simulating the shadows cast by buildings and trees according to the position of the sun. 
Starting from a classified LiDAR point cloud, the internship will focus on developing a ray-tracing algorithm dedicated to urban shadow computation,
incorporating both temporal and seasonal parameters.

The project will also include a strong emphasis on documentation and reproducibility: writing methodological guides, structuring data according 
to FAIR principles, and contributing to open-source documentation to enable the reuse of the processing pipeline in other territories. 
Particular attention will be paid to reproducibility, computational efficiency, and the transferability of the methods across different 
territories.
